# 介绍
一个将 Windows 系统下常用快捷键转变成 MacOS 的脚本，你只需要将 mac.ahk 文件编译（或者使用我上传的 exe）设置为开机自启动就可以了。

这个脚本主要是帮助一些像我这样长期使用 MacOS 但有时候又不得不使用 Windows 系统来处理一些问题的人使用。

# 现状
目前实现了 复制、粘贴、全选、输入法切换 等常用快捷键，具体可以见 mac.ahk 文件的内容。
